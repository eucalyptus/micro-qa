Diese Spalte zeigt die Beschreibung eines Projekts an (HTML formatiert).

Mit der Option "Projektname anzeigen" kann man den Projektnamen oberhalb der Beschreibung anzeigen lassen.
Mit der Option "Beschreibung begrenzen" kann die Beschreibung auf eine bestimmte Anzahl von Zeilen beschränkt werden.
Die Einschränkung erfolgt anhand der <br/> Tags.
